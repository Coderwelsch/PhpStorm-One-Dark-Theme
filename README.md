# PhpStorm-One-Dark-Theme
A better One Dark (Atom) syntax theme for PhpStorm and the IntelliJ IDE family.

## Language Support
Currently only these languages are supported:
- CSS
- Sass/SCSS
- JSON
- Handlebars/Mustache
- Javascript
- Markdown
